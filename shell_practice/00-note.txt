Shell Scpring:
==============

1) Printing value of 3+3 --> echo `expr 3 + 3`
2) Printing "Hello - (value of) 3+3" --> echo "Hello - `expr 3 + 3`"
3) Exit Status of the previous executed program can be got using: "$?"
